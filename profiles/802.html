Summary of Application Experience

**~~***~~~***~~~***~~~***~~~***~~~***~~~***~~~***EXPENSES***~~~***~~~***~~~***~~~***~~~***~~~***~~~***~~**
_____________________________________________________________________________________________

MCAT REVIEW BOOKS: $364 for TBR and $210 for a Mix of Others, and $130 for Prep Class Materials = $704

This was a bad idea on my part because I ended up getting almost all books people recommended but only ended up using less than half of them -_- I suggest people get only one set of company books and really focus on that because now I'm having trouble selling mine.

PREP CLASS (Pay As You Go So Didn't Attend All sessions): $420

I was running out of money so by the end of the course I only attended sessions where they had topics I needed help on.

AAMC PRACTICE EXAMS: $245

Seriously though, why are these practice exams so expensive?

ACTUAL MCAT: $290

I ended up postponing my MCAT so yeah...

PRIMARY APPLICATION FEES: $390

I am applying as a disadvantaged applicant so that saved me a lot of money since I didn't have to pay for any secondaries out of pocket. This was the reason I didn't complete a secondary for Maryland because it was the only school on the list which didn't accept the fee waiver.

ATTIRE: $150

I actually got a good deal on my suit and shoes! The blouse was pretty expensive though.

LODGING: $125 + $96 + $52 = $273

3 hotel/motel costs so far and have more on the way so this will keep increasing.

TOTAL: $2472, so far. Why is this application process so damn expensive. I hope to get an acceptance soon =[ I don't think I can afford to go through these expenses for another cycle. *Sigh* Hoping for the best!
